快速上手

安装
$ npm install -g @humanclaw/humanclaw
启动服务
$ humanclaw serve
# Dashboard: http://localhost:2026
注册碳基节点
$ humanclaw agent add
# 交互式录入：节点名称、技能标签
AI 规划任务
$ humanclaw plan "完成首页重构"
# AI 自动拆解任务、匹配节点、生成话术
